# telegram_bot

Bot de Telegram para agency skokkers

## Correr el proyecto

1. Crear un entorno virtual:

    ```sh
    python -m venv env
    ```

2. Activar el entorno virtual:

    ```sh
    source env/bin/activate
    ```

3. Instalar las dependencias:

    ```sh
    pip install -r requirements.txt
    ```

4. Ejecutar el bot:

    ```sh
    python src/main.py
    ```
